Configuración del correo remitente (Microsoft 365)
Para enviar correos desde el sistema necesitas conectar una cuenta de Microsoft 365 vía Azure App Registration con OAuth2.
Pasos: (1) En Azure Portal → App Registrations → Nueva app. (2) Permisos: Mail.Send en Microsoft Graph. (3) Generar Client Secret. (4) Ingresar los datos abajo.
La cuenta remitente puede ser exclusiva como cobranza@dschillan.cl.
